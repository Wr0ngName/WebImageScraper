# WebScraper
## Requirements
 * Python 3.7
 * PIP
 * Internet connection
 
## Installation
 Run `python -m pip install -r .\requires.txt` to install libraries
 
## Run
 Use `python scraper.py <url>` to get started. Type `python scraper.py help` for full details about the parameters.
